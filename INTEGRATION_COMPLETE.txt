## 🔐 Stripe Integration (Next Steps)

### Get Stripe Publishable Key
1. Go to: https://dashboard.stripe.com/apikeys
2. Copy "Publishable key" (starts with `pk_live_`)
3. Update in both environment files:

**src/environments/environment.ts**
```typescript
stripe: {
  publishableKey: 'pk_live_your_actual_key_here'
}
```

**src/environments/environment.prod.ts** (same)

### Implement Payment Processing

In `booking.component.ts`, update the `onSubmit()` method:

```typescript
import { loadStripe } from '@stripe/stripe-js';

async onSubmit() {
  if (this.bookingForm.invalid || !this.listing) return;
  
  this.processing = true;
  const stripe = await loadStripe(environment.stripe.publishableKey);
  
  // Create booking first
  const booking = { /* ... */ };
  
  this.supabaseService.createBooking(booking)
    .subscribe(async savedBooking => {
      // Call your backend endpoint to create Stripe session
      const response = await fetch('/api/create-checkout-session', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          bookingId: savedBooking.id,
          amount: this.totalPrice * 100,
          email: this.bookingForm.get('email')!.value
        })
      });
      
      const { sessionId } = await response.json();
      await stripe?.redirectToCheckout({ sessionId });
    });
}
```

### Create Netlify Function for Payment

Create: `netlify/functions/create-checkout-session.ts`

```typescript
import Stripe from 'stripe';

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!);

export default async (req: any) => {
  if (req.method !== 'POST') {
    return { statusCode: 405, body: 'Method Not Allowed' };
  }

  const { bookingId, amount, email } = JSON.parse(req.body);

  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: [
      {
        price_data: {
          currency: 'eur',
          product_data: { name: 'Property Booking' },
          unit_amount: amount,
        },
        quantity: 1,
      },
    ],
    mode: 'payment',
    success_url: `${process.env.URL}/booking-success?session_id={CHECKOUT_SESSION_ID}`,
    cancel_url: `${process.env.URL}/bookings`,
    customer_email: email,
    metadata: { bookingId },
  });

  return {
    statusCode: 200,
    body: JSON.stringify({ sessionId: session.id }),
  };
};
```

### Add Netlify Env Variables

1. Go to Netlify site → Settings → Environment
2. Add:
   ```
   STRIPE_SECRET_KEY = sk_live_your_secret_key
   STRIPE_WEBHOOK_SECRET = whsec_your_webhook_secret
   ```

## 🛡️ Security Best Practices

1. **Never commit secrets** to git — use Netlify env vars
2. **Row Level Security (RLS)**: Optional but recommended for production
   - Listings: Public read-only
   - Bookings: Users see only their own

3. **API Key**: Using `anonKey` is safe for client-side queries on public data

4. **Stripe Webhooks**: Set up for payment confirmation

## ❓ Troubleshooting

| Issue | Solution |
|---|---|
| "Cannot find listings" | Run sql-migration.sql in Supabase SQL Editor |
| CORS error | Supabase has CORS enabled by default; if issue persists, check Supabase settings |
| Stripe not loading | Verify publishable key is correct in environment file |
| Bookings not saving | Check Netlify function logs or Supabase database |
